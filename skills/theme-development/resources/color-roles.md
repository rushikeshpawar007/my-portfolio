# Color Roles Reference

Quick reference for which CSS variable to use for each design role.

## Primary Colors
| Role | Variable | Light | Dark |
|------|----------|-------|------|
| Page background | `--bg-color` | #ffffff | #111827 |
| Body text | `--text-primary` | #0f172a | #eaeaea |
| Supporting text | `--text-color` | #657b83 | #d1d5db |
| Headings | `--heading-color` | #0b1220 | #f9fafb |

## Accent Colors
| Role | Variable | Light | Dark |
|------|----------|-------|------|
| Primary accent | `--accent-color` | #0d9488 | #2dd4bf |
| Accent hover | `--accent-hover` | #0f766e | #5eead4 |
| Accent RGB | `--accent-rgb` | 13,148,136 | 45,212,191 |
| Secondary accent | `--accent-secondary` | #f59e0b | #f59e0b |

## Surface Colors
| Role | Variable | Usage |
|------|----------|-------|
| Glass background | `--glass-bg` | Card surfaces |
| Glass border | `--glass-border` | Card borders, dividers |
| Form background | `--form-bg` | Input fields |

## Effect Colors
| Role | Variable | Usage |
|------|----------|-------|
| Hero glow | `--hero-glow` | Radial gradient behind hero |
| Card glow | `--card-glow` | Hover glow on cards |
| Pulse | `--pulse-color` | Animation pulse effects |
